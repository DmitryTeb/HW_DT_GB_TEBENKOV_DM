# Credit Default Prediction

## Описание проекта

Цель проекта — построить модель машинного обучения для прогнозирования вероятности невыполнения клиентом кредитных обязательств (Credit Default) на основе данных о финансовом состоянии заемщиков.

В рамках работы реализована модель классификации, обученная на исторических данных клиентов банка. Модель используется для предсказания дефолта на тестовом наборе данных.

Основной метрикой оценки качества является **F1-score для класса 1 (кредитный дефолт)**.

# Feature Engineering

Для улучшения качества модели были созданы дополнительные признаки.

## Debt-to-Income (DTI)

Показатель долговой нагрузки клиента.

DTI = Monthly Debt / (Annual Income / 12)


Этот показатель отражает долю ежемесячного дохода, направляемую на обслуживание долгов.

---

## Negative Records

Агрегированный показатель кредитных проблем клиента.

Формируется как сумма:

- Number of Credit Problems  
- Bankruptcies  
- Tax Liens  

Этот признак отражает общее количество негативных записей в кредитной истории.

# Модели

В рамках проекта были обучены и сравнены следующие алгоритмы машинного обучения:

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Метрики оценки

Использовались следующие метрики:

- **F1-score**
- **AUC**
- **Gini**

Формула Gini:

Gini = 2 * AUC - 1
